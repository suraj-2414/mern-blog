# MERN Blog App

A full-stack blog application built using React.js, Node.js, Express.js, MongoDB, and JWT Authentication.

## Features

* User Registration and Login
* JWT Authentication
* Protected Routes
* Create Blog Posts
* Read Blog Posts
* Category-wise Blog Display

## Tech Stack

### Frontend

* React.js
* React Router DOM
* CSS

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Mongoose

## Project Structure

BLOG APP

├── FRONTEND

└── BACKEND

## Installation

### Clone Repository

git clone https://github.com/suraj-2414/mern-blog.git

### Backend Setup

cd BACKEND

npm install

npm start

### Frontend Setup

cd FRONTEND

npm install

npm run dev
